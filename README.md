# Bellabeat Case Study

## Project Overview

This project analyzes Fitbit fitness tracker data to identify patterns in users' physical activity and sleep habits. The goal is to generate business insights that could help Bellabeat improve its smart wellness products.

## Business Question

How can activity and sleep data help Bellabeat better understand user behavior and improve customer experience?

## Tools Used

- R
- tidyverse
- lubridate
- ggplot2
- R Markdown

## Data Source

Fitbit Fitness Tracker Data (public dataset provided for educational purposes).

## Analysis Process

- Data import
- Data cleaning
- Missing values analysis
- Duplicate detection
- Date formatting
- Exploratory Data Analysis (EDA)
- Data visualization
- Business recommendations

## Key Findings

- Physical activity varies across the days of the week.
- Daily steps have a moderate positive correlation with calories burned.
- Users spend a significant amount of time in sedentary behavior.
- Average sleep duration is approximately 7.1 hours.
- Sleep duration alone shows a weak relationship with daily step count.

## Business Recommendations

- Encourage users to increase activity during less active days.
- Implement reminders to reduce sedentary time.
- Promote healthy sleep habits through personalized notifications.
- Combine activity and sleep metrics to provide personalized recommendations.
